# CSPSolver
Программа для решения задачи линейного раскроя, разработанная для предприятия по производству окон.
Программа не является готовым универсальным решением задачи линейного раскроя, поскольку была предназначена для использования в рамках конкретного предприятия с узкоспециализированным оборудованием. Приведённый в репозитории код выложен как пример реализации алгоритма линейного раскроя на языке Java.

## Сборка
Для сборки программы применяется Gradle.
Скопируйте файлы программы и соберите проект, используя команду `gradle build`.

## Использование
Запуск программы осуществляется через собранный JAR-артефакт.
После открытия JAR-файла отобразится окно выбора файла. Выберите Excel-файл, содержание которого аналогично содержанию одного из Excel-файлов, находящихся в папке examples.
Программа обработает Excel-файл и создаст новый текстовый файл в той же папке, в которой находится JAR-файл.
Итоговый текстовый файл содержат требуемую информацию для запуска станка, использующего формат вводных данных, аналогичный указанному в текстовом файле.
